## Common Mistakes to Avoid

### ‚ùå Copying technical styles between concepts

```
Valentine used fish-eye ‚Üí Don't auto-apply to Halloween
Christmas has warm tones ‚Üí Don't auto-apply to Summer
```

### ‚ùå Mentioning identity

```
"Beautiful blonde woman" ‚Üí WRONG
"Warm genuine smile" ‚Üí CORRECT
```

### ‚ùå Forgetting CRITICAL tags

```
If the red fur backdrop is essential, tag it CRITICAL
Otherwise model might substitute something "realistic"
```

### ‚ùå All prompts looking the same

```
If all 8 prompts have same lighting + pose + aesthetic = FAIL
Must have variety across axes
```
