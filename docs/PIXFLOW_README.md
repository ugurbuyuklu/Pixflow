# Pixflow - Desktop AI Asset Pipeline

Pixflow is the current desktop product. It focuses on UI/UX-driven workflows for creating marketing assets.

## Core flows

1. Prompt Factory
2. Asset Monster
3. Avatar Studio
4. The Machine
5. Library

## Active architecture

- Electron host + embedded Express API + React renderer
- Runtime outputs stored locally (`outputs/`, `uploads/`, `data/`)

## Legacy archive

Previous-version materials are stored under:
- `Burgflow Archive/`

Do not add new Pixflow work into archive folders.

## Documentation language policy

- All active documentation must be written in English.
- If a non-English document is found in active folders, translate it to English and keep English as the source of truth.
